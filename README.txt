

사용방법



1. 실행하기 위해선 파이선 가상환경이 필요합니다. 가상환경 설치는 Venv_installer.py에서 합니다.
2. executor.py 가 실행파일입니다.
3. 실행 시 카메라가 연결 되어있어야 합니다.
4. 실행 시 *.pt 파일이 source 폴더 내에 존재하여야 합니다. 테스트용 *.pt 파일이 첨부되어 있습니다.
5. *.pt 파일의 이름은 ./source/define.txt 파일에서 확인 및 변경 가능합니다. 
6. 사진 전송 기능 이용시 서버 연결이 필수입니다.
7. 서버 연결 시 대상 서버의 아이피와 포트는 ./source/define.txt 파일에서 확인 및 변경 가능합니다. (현재 테스트 용으로 127.0.0.1;8080 으로 되어있습니다.)
8. 서버 구동 프로그램은 ./server/tcp_server.py 입니다.
