

사용방법



1. 실행하기 위해선 파이선 가상환경이 필요합니다. 가상환경 설치는 Venv_installer.py에서 합니다.
2. executor.py 가 실행파일입니다.
3. 작동 시 카메라가 연결 되어있어야 합니다.
4. 작동 시 *.h5 파일이 source 폴더 내에 존재하여야 합니다. (파일이 너무 커 첨부하지 못했습니다. *.h5 파일의 이름은 classifier.txt 에서 확인 및 변경 가능합니다.)
5. 서버 연결 시 대상 서버의 아이피와 포트는 ./source/IPnPort.txt 파일에서 확인 및 변경 가능합니다. (현재 테스트 용으로 127.0.0.1;8080 으로 되어있습니다.)
6. 서버 구동 프로그램은 tcp_server.py 입니다.

7. 테스트용 *.h5파일 공유 gdrive 주소 : https://drive.google.com/file/d/1Yzg8DHc056H2gU_GNsZ-y-Cz92O8gtmT/view?usp=sharing
